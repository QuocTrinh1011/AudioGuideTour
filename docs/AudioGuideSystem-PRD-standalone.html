PRD tái thiết kế theo codebase thực tế

AudioGuideSystem: PRD chi tiết cho hệ thống audio tour, QR, geofence và quản trị nội dung du lịch

Tài liệu này thay thế cách kể chuyện cũ trong file mẫu bằng hiện trạng thật của project đang có: AudioGuideAdmin là CMS vận hành, AudioGuideAPI là backend runtime/public API, và AudioTourApp là ứng dụng MAUI Android cho visitor. Mọi mô tả, quy tắc và sequence dưới đây đều bám theo controller, page, service và model hiện có trong repo.

ASP.NET Core MVC Admin .NET 8 Web API .NET MAUI Android Geofence + Audio Queue QR Lookup + Camera Scan Shared DB + Shared Storage Session Auth cho Admin
PRD standalone được dựng từ codebase tại thời điểm hiện tại của workspace AudioGuideSystem.
